TreePer

This is a little, quick and dirty project. I've integrated the work of Yannis Gravezas inside YouTube using a Chrome extension to avoid CORS and TOS violations.
More info on github.com

Usage

Just click the tree icon that appears on the youtube player controls bar. You can also use other keyboard controls.

Share

You can share the TreePer experience with friends sending them the youtube link. They will see the same visual effect with the same opacity.

Controls

Use the keyobard.
Button Action
[ , - Decrease video opacity (animation will start automatically)
] , + Increase video opacity (animation will stop automatically)
b, B Previous shader
n, N Next shader
